# ☕ Repositório de Estudos — Java

Repositório de aprendizado desenvolvido durante curso presencial 
de Java, em 2022. Reúne exercícios e projetos práticos 
construídos ao longo do curso, com foco em Programação Orientada 
a Objetos.

## Projetos

- **Agenda** — sistema de agenda com operações básicas
- **Cadastro** — sistema de cadastro simples com interface
- **Calculadora de Área** — exercício de UML aplicado
- **Calculadora IMC** — cálculo de índice de massa corporal
- **ProjectManager** — gerenciador de projetos inspirado no Trello
- **RealProduct** — primeiros exercícios com POO
- **Herança, Polimorfismo e Abstração** — exercício de revisão geral

## Contexto

Esses projetos marcam meu primeiro contato sério com programação 
orientada a objetos. Cada um foi construído do zero, com atenção 
à documentação e organização do código — um hábito que carrego 
até hoje.
